# \# CSV Data Cleaning and Automation

# 

A Python script that takes messy CSV sales data, cleans it automatically, 

and generates a summary report by region and month.



\## What it does

\- Removes rows with missing names, sales, or months

\- Removes invalid non-numeric sales values

\- Generates total, average, highest and lowest sales summary

\- Saves cleaned data to a new CSV file



\## Libraries Used

\- pandas

\- openpyxl



\## How to Run

pip install pandas openpyxl

python create\_data.py

python clean\_data.py
